FileShrinkr is a free online image toolkit built by DSnM Solutions. We help creators, small businesses, and everyday users shrink photo file sizes, export PDFs, and make quick edits without installing desktop software.

What makes FileShrinkr different

Many compressors only handle a single format. FileShrinkr supports HEIC (iPhone photos), AVIF, PNG, JPG, WEBP, and PDF output — with optional target file sizes (20–500 KB), social resize presets, and batch ZIP downloads. Browser (private) mode keeps JPG, WEBP, AVIF, and PNG compression on your device; server mode handles PDF, HEIC conversion, EXIF removal, and MozJPEG JPG optimization.

Our approach

We use Pillow (and MozJPEG when available) with img2pdf for PDF workflows. No account is required. We focus on practical workflows: government form file limits, email attachments, website performance, WhatsApp sharing, Instagram sizing, and document assembly from scans or photos.

FileShrinkr is operated by DSnM Solutions. For questions, visit our Contact page.
