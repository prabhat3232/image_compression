What you can do with FileShrinkr

  • Compress images — JPG, PNG, WEBP, HEIC, and SVG in; JPG, WEBP, AVIF, PNG, or PDF out. Browser-private mode, quality slider, target sizes (20–500 KB) for government and exam form uploads, and Instagram/Facebook/YouTube presets.
  • Resize and edit for forms — Crop passport-style photos, set custom width × height, and compress to portal file-size limits before filling SSC, UPSC, railway, or other online application forms. Read the full guide.
  • Convert to PDF — Export compressed images as PDF, or merge multiple images into one document (lossless option available).
  • Batch processing — Upload multiple files; download a ZIP when compressing several images.
  • Edit in browser — Crop, resize, rotate, and flip client-side, then hand off to the compressor.
  • Before/after preview — Compare original and compressed images with a slider after processing.

How to compress images online

  1. Choose Browser (private) or Server mode, set quality or a target file size, and pick a resize preset if needed.
  2. Select output format: JPG, WEBP, AVIF, PNG, or PDF. Enable EXIF removal or MozJPEG (server) when required.
  3. Click Start Compression — preview before/after, then download automatically.
